## Введение

Данный документ описывает план тестирования веб-приложения **BookFinder** — платформы для поиска, просмотра и управления списком избранных книг.

Цель тестирования — проверить корректность реализации всех заявленных функций (регистрация, авторизация, поиск книг, добавление в избранное, работа фильтров и динамического поиска), а также убедиться в удобстве и стабильности интерфейса.

Тестирование проводится вручную с проверкой взаимодействия между фронтендом (Thymeleaf / React) и серверной частью (Spring Boot, REST API, MySQL).

---

## Объект тестирования

Объектом тестирования является веб-приложение **BookFinder**, реализующее следующие основные функции:

- Авторизация и регистрация пользователей  
- Поиск и фильтрация книг по названию и автору  
- Просмотр информации о книге  
- Добавление и удаление книг из избранного  
- Отображение списка избранных книг  
- Корректная работа бокового меню и выхода из аккаунта  

---

## Атрибуты качества

### 1. Функциональность
- **Полнота:** реализованы все заявленные функции (регистрация, логин, поиск, избранное).  
- **Корректность:** добавление/удаление из избранного изменяет состояние книги только у текущего пользователя.  

### 2. Удобство использования
- Динамический поиск должен работать без перезагрузки страницы.  
- Иконка «звездочки» должна отображать состояние избранного и реагировать мгновенно.  
- Интерфейс должен быть понятным и адаптивным.  

### 3. Надежность
- Все операции выполняются корректно при одновременном доступе нескольких пользователей.  
- Отсутствие сбоев при работе с базой данных.  

---

## Риски

Возможные риски при тестировании и эксплуатации:

- Потеря состояния избранных книг при сбое БД.  
- Ошибки при обновлении AJAX-запросов (из-за сетевых задержек).  
- Некорректная фильтрация при использовании спецсимволов в поиске.  
- Несоответствие отображения состояния кнопки избранного текущему состоянию в БД.  

---

## Аспекты тестирования

В процессе тестирования будут проверены следующие аспекты работы приложения:

- Регистрация и вход пользователя  
- Динамический поиск книг по названию и автору  
- Открытие страницы с подробной информацией о книге  
- Добавление и удаление книги из избранного (на странице книги и в общем списке)  
- Отображение корректного состояния звездочки (активна/неактивна)  
- Просмотр списка избранных книг  
- Корректная работа кнопки «Выйти» без перехода на страницу логина  

---

### Тест: регистрация и вход

**Проверить:**
- успешную регистрацию нового пользователя;  
- авторизацию с корректными данными;  
- сообщение об ошибке при неверном пароле.  

---

### Тест: поиск и фильтрация

**Проверить:**
- ввод названия книги — в списке остаются только совпадения;  
- ввод имени автора — поиск корректен;  
- поиск без учёта регистра;  
- отсутствие результатов выводит пустой список.  

---

### Тест: добавление в избранное

**Проверить:**
- клик по незакрашенной звездочке добавляет книгу в избранное;  
- клик по закрашенной звездочке удаляет из избранного;  
- обновление страницы сохраняет состояние звездочек;  
- у разных пользователей — разные списки избранного.  

---

### Тест: просмотр книги

**Проверить:**
- корректное отображение информации (название, автор, жанр);  
- правильное состояние кнопки избранного.  

---

### Тест: выход из аккаунта

**Проверить:**
- при нажатии «Выйти» пользователь остаётся на главной странице;  
- авторизация сбрасывается;  
- звездочки и кнопки избранного становятся неактивными.  

---

## Подходы к тестированию

Будет использован **ручной подход**:

- тестирование функциональности через браузер (Chrome, Firefox);  
- проверка AJAX-запросов через DevTools → Network;  
- проверка данных в базе MySQL для подтверждения корректности изменений.
